Source-chan é um bot para Discord criado atravez da comunidade [Source BR](https://www.sourcebr.com.br/).
**Ela não é desenvolvida pela Valve nem possui ligação oficial** — mas nasceu da paixão pela engine que marcou gerações de jogadores e modders.

O objetivo da Source-chan é simples: trazer a essência da Source Engine para o Discord, combinando funções úteis, diversão e um toque nostálgico para quem cresceu ou ainda vive no universo dos jogos da Valve.

---

## O que é a Source-chan?
-  Um bot inspirado na Source Engine, feito de fã para fãs.
-  Criada para o servidor [Source BR](https://discord.gg/tVNv6SNZZT), mas disponível para todos.
-  Modular, aberta a contribuições e em constante evolução.
-  Construída em [Node.js](https://nodejs.org/pt) com [Discord.js](https://discord.js.org/).
-  Código aberto para que qualquer um possa melhorar, adaptar e expandir.
